How to Test Synchronization
  1. Open GrooveScribe: Click "Open Main GrooveScribe" button above
  2. Create or load a groove: Add some notes to the staff
  3. Access Visual Sync: Click the metronome options menu (gear icon next to metronome buttons)
  4. Select "Visual sync": This should open the synchronization adjustment popup
  5. Play the groove: Use the play button to start playback
  6. Observe synchronization: Watch how the visual highlighting moves with the audio
  7. Adjust if needed: Use the slider in the Visual Sync popup to fine-tune timing

Expected behavior:

  • Visual highlighting should move in sync with the audio playback
  • The default 50ms offset should compensate for typical audio latency
  • Adjusting the offset should immediately affect the synchronization
  • Different offset values should produce noticeable timing differences
