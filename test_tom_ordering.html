Explanation

Why this mapping?

  • Tom 1 (top) = High rack tom - highest pitch
  • Tom 4 (middle) = Low rack tom - medium pitch (was confusingly labeled "Tom 2")
  • Tom 3 (bottom) = Floor tom - lowest pitch (was labeled "Floor Tom")

Note: There is no "Tom 2" function in the code - the system uses tom1, tom2, and tom4 functions, but the labels now correctly reflect the logical drum numbering from high to low pitch.
